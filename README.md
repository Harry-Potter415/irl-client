README
======

React/Redux frontend hooked up to irl-server as an API

## Installationn

Make sure you're on Node v8.9.0 with npm v6.7.0

Install packages
`npm install`

Configure environment variables
`cp .env.sample .env`

## Running App

`npm run start`

The client is now running on http://localhost:3001
